# devise.cr

NOT a devise remake. Helpers to get session info from Rails+Devise sessions.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     devise:
       github: public-law/devise.cr
   ```

2. Run `shards install`
